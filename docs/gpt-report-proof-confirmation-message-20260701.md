프로젝트: `perpackage-cafe24-file-upload-app`

## 1. 작업 목적

`/admin?file_id=<file_id>` 파일 상세 화면에 제작 전 고객에게 보낼 `교정확인 안내문` 생성 및 복사 기능을 추가했다.

운영자가 파일 정보를 확인한 뒤 채널톡, 카카오톡, 이메일 등 기존 응대 채널로 교정확인 안내문을 바로 전달할 수 있게 하는 관리자 편의 기능이다.

이번 작업에서는 DB 변경, 상태값 추가, 고객용 교정확인 링크, 자동 발송, 파일 자동 승인 처리는 하지 않았다.

## 4. 교정확인 안내문 생성 방식

안내문은 브라우저 상태값을 기반으로 생성한다.

반영 데이터:

- `files.order_id` 또는 `미연결`
- `files.original_filename`
- `files.id`
- 선택한 교정확인 항목
- 관리자가 입력한 추가 확인 메모

기본 구조:

```txt
안녕하세요. 페르패키지입니다.

전달주신 인쇄 파일 기준으로 제작 전 교정 확인 안내드립니다.

주문번호: {order_id 또는 미연결}
파일명: {original_filename}
업로드 파일 ID: {file_id}

아래 내용 확인 부탁드립니다.

{번호 목록}

추가 확인 사항: {관리자 입력 메모}

교정 내용 확인 후 "확인했습니다"라고 회신 주시면 이후 제작 진행 단계로 넘어가겠습니다.

수정이 필요한 부분이 있다면 함께 말씀 부탁드립니다.

감사합니다.
```

추가 확인 메모가 비어 있으면 `추가 확인 사항` 줄은 표시하지 않는다.

## 5. 프리셋 선택/해제 방식

추가한 프리셋:

| 버튼 라벨 | 안내문 반영 문구 |
| --- | --- |
| 오탈자 확인 필요 | 오탈자 여부를 확인해 주세요. |
| 로고 위치 확인 필요 | 로고 위치가 의도하신 위치와 맞는지 확인해 주세요. |
| 칼선 기준 위치 확인 | 칼선 기준으로 디자인 위치가 맞는지 확인해 주세요. |
| 색상 참고 사항 | 모니터 색상과 실제 인쇄 색상은 차이가 있을 수 있습니다. |
| 후가공 위치 확인 | 후가공 위치가 의도하신 위치와 맞는지 확인해 주세요. |
| 수량/사이즈 확인 | 주문 수량과 제작 사이즈를 다시 확인해 주세요. |
| 접힘/뚜껑 방향 확인 | 박스 접힘 방향과 뚜껑 방향을 확인해 주세요. |

동작 방식:

- 버튼 클릭 시 해당 항목을 선택한다.
- 이미 선택된 버튼을 다시 클릭하면 선택 해제한다.
- 여러 항목을 동시에 선택할 수 있다.
- 선택된 항목은 안내문에 `1.`, `2.`, `3.` 번호 목록으로 반영된다.
- 선택 항목이 없으면 기본 확인 목록을 표시한다.

기본 확인 목록:

```txt
1. 인쇄 내용
2. 오탈자
3. 로고/이미지 위치
4. 칼선 기준 위치
5. 색상 참고 사항
```

## 6. 복사 기능 구현 방식

`navigator.clipboard.writeText()`를 사용해 교정확인 안내문 전체를 클립보드에 복사한다.

피드백:

- 성공: `복사되었습니다.`
- 실패: `복사에 실패했습니다. 안내문을 직접 선택해 복사해 주세요.`

클립보드 복사가 막히는 환경에서도 안내문은 readonly textarea로 표시되어 직접 선택 복사할 수 있다.

## 14. 운영자가 `/admin`에서 확인해야 할 테스트 순서

1. `/admin` 로그인
2. `file_id` 검색
3. 파일 상세 화면에서 `교정확인 안내문` 섹션 확인
4. 교정확인 프리셋 버튼 7개 표시 확인
5. 버튼 클릭 시 선택 상태가 바뀌는지 확인
6. 여러 버튼을 동시에 선택할 수 있는지 확인
7. 선택한 항목이 고객 안내문에 번호 목록으로 반영되는지 확인
8. 아무 항목도 선택하지 않았을 때 기본 확인 목록이 표시되는지 확인
9. 추가 확인 메모 입력 시 안내문에 `추가 확인 사항`으로 반영되는지 확인
10. `교정확인 안내문 복사` 버튼이 동작하는지 확인
11. 복사 성공 피드백이 표시되는지 확인
12. 재업로드 요청 안내문 기능이 기존처럼 유지되는지 확인
13. 파일 상태 변경, 주문번호 연결 이력, 다운로드 버튼, Webhook 로그 필터가 유지되는지 확인
