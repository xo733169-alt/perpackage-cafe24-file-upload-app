- 작성 기준: 2026-07-06
- 대상 화면: Cafe24 고객 마이쇼핑 좌측 고객센터/FAQ 메뉴
- 문서 성격: UI 개선 전 메뉴 점검 결과

## 1. 문서 목적

이 문서는 Cafe24 마이쇼핑 좌측 고객센터/FAQ 메뉴를 실제 운영 기준으로 정리하기 위한 점검 결과 문서입니다.

목적은 다음과 같습니다.

- 마이쇼핑 좌측 고객센터/FAQ 메뉴 중 실제 운영에 필요한 메뉴와 숨김 후보를 구분합니다.
- 작동하지 않거나 현재 페르패키지 운영과 맞지 않는 메뉴를 정리합니다.
- 메뉴를 바로 삭제하지 않고 숨김 후보로 관리합니다.
- 주문, 회원, 배송, 취소/반품/교환 관련 메뉴는 기능 확인 후 판단합니다.

## 3. 유지 후보 정리

아래 메뉴는 유지 후보입니다.

- 네이버톡톡 상담
- 카카오톡채널
- 자주묻는질문, 단 내용 최신화 필요
- 주문/결제/영수증
- 배송/상품/상품평
- 취소/반품/교환
- 아이디/비밀번호 찾기
- 주문/배송조회
- 회원정보수정

유지 후보라도 문구, 위치, 내용은 개선할 수 있습니다. 단, 주문/회원/배송 기능과 연결된 메뉴는 기능이 깨지지 않게 링크와 동작을 먼저 확인합니다.

## 4. 숨김 후보 정리

아래 메뉴는 숨김 후보입니다.

- 이벤트/제휴/기타
- 이달의 베스트 후기
- 상품입고일정
- 해외배송문의, 해외배송 미운영 시
- 직접추가가능
- 오프라인 매장안내, 실제 안내 페이지가 없거나 최신화되지 않은 경우
- 대량구매문의 관리, 실제 사용하지 않는 경우
- 좋아요, 실제 운영하지 않는 경우

숨김 후보는 삭제 대상이 아닙니다. 실제 사용 여부, 링크 동작, 운영 필요성을 확인한 뒤 숨김 처리 여부를 결정합니다.

## 5. 중복 가능 메뉴 정리

아래 메뉴는 서로 역할이 겹칠 수 있으므로 정리 전 확인이 필요합니다.

| 중복 가능 메뉴 | 확인할 내용 | 정리 방향 |
|---|---|---|
| 상품문의하기 / 상품문의 | 같은 게시판으로 이동하는지, 서로 다른 문의 흐름인지 확인 | 같은 기능이면 하나로 통합 또는 한쪽 숨김 검토 |
| 주문조회 / 주문/배송조회 | 같은 주문조회 화면인지, 배송조회 전용 기능인지 확인 | 핵심 주문조회 경로는 유지 |
| 회원정보 / 회원정보수정 | 같은 회원정보 수정 화면인지 확인 | 고객이 이해하기 쉬운 이름으로 정리 |
| 네이버톡톡 상담 / 카카오톡채널 / 폼메일 상담 | 실제 사용하는 상담 채널과 미사용 채널 구분 | 운영 채널 중심으로 정리 |
| 게시물관리 / 상품문의 / FAQ 게시판 | 고객이 작성한 글 관리와 새 문의 작성이 섞여 있는지 확인 | 문의 작성, 문의 내역, FAQ를 분리해 안내 |

## 6. 실제 개선 방향 제안

- 좌측 고객센터 영역은 “상담 채널”과 “FAQ”로 단순화합니다.
- 상담 채널은 네이버톡톡, 카카오톡채널, 폼메일 상담만 남길지 검토합니다.
- FAQ 카테고리는 실제 운영 내용이 있는 것만 남깁니다.
- `직접추가가능` 같은 샘플 메뉴는 숨김 후보로 둡니다.
- 주문, 회원, 배송 관련 기능 링크는 유지합니다.
- 사용하지 않는 메뉴는 삭제하지 말고 숨김 처리부터 검토합니다.
- 고객이 파일 업로드 상태를 찾는 흐름은 주문상세 안내 박스와 `/file-status` 중심으로 유지합니다.

## 7. 실제 수정 전 체크리스트

- [ ] 각 메뉴 클릭 시 이동 URL 확인
- [ ] 실제 내용이 있는지 확인
- [ ] 고객이 실제 사용하는 메뉴인지 확인
- [ ] 운영자가 관리하는 게시판인지 확인
- [ ] 숨김 후보와 유지 후보 구분
- [ ] 숨김 전 원본 코드 백업
- [ ] PC 화면 확인
- [ ] 모바일 화면 확인
- [ ] 숨김 후 주문조회/회원/배송 기능 정상 여부 확인

## 8. 안전 기준

- 바로 삭제하지 않습니다.
- 숨김 전 링크와 기능을 확인합니다.
- `module` 속성은 수정하지 않습니다.
- `{$변수}` 형태의 Cafe24 템플릿 변수는 수정하지 않습니다.
- `onclick` 동작은 수정하지 않습니다.
- 주문조회, 회원정보, 배송주소록, 취소/반품/교환 기능은 삭제하지 않습니다.
- `display:none` 또는 주석 처리 전 원본 백업을 남깁니다.
- PC와 모바일을 모두 확인합니다.
